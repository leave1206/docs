## Request

```bash
curl https://llm.bytespike.ai/v1/images/generations \
  -H "x-api-key: $BYTESPIKE_API_KEY" \
  -H "content-type: application/json" \
  -d '{
    "model": "seedream-v5lite",
    "prompt": "An ink-wash mountain at dawn, soft mist",
    "size": "1024x1024",
    "n": 4
  }'
```

### Body 参数

| Field | Type | Required | Default | Notes |
|---|---|---|---|---|
| `model` | string | yes | — | `seedream-v5lite` |
| `prompt` | string | yes | — | 原生接受 CJK。 |
| `size` | string | no | `1024x1024` | 支持：`1024x1024`、`1024x1536`、`1536x1024`。 |
| `n` | integer | no | 1 | 1–4。每张单独计费。 |
| `seed` | integer | no | — | 支持可复现性。 |
| `negative_prompt` | string | no | — | 标准 SD 风格的反向 prompt。 |

## Response

```json
{
  "created": 1715174400,
  "data": [{"url": "https://cdn.bytespike.ai/img/...", "revised_prompt": "..."}]
}
```

URL 预签名，24h 有效。

## 代码示例

<CodeGroup>

```bash cURL
curl https://llm.bytespike.ai/v1/images/generations \
  -H "x-api-key: $BYTESPIKE_API_KEY" \
  -H "content-type: application/json" \
  -d '{"model": "seedream-v5lite", "prompt": "An ink-wash mountain at dawn", "n": 4}'
```

```python Python
from openai import OpenAI

client = OpenAI(base_url="https://llm.bytespike.ai/v1", api_key="$BYTESPIKE_API_KEY")
resp = client.images.generate(
    model="seedream-v5lite",
    prompt="An ink-wash mountain at dawn, soft mist",
    size="1024x1024",
    n=4,
)
for img in resp.data:
    print(img.url)
```

```javascript Node
import OpenAI from "openai"

const client = new OpenAI({
  baseURL: "https://llm.bytespike.ai/v1",
  apiKey: process.env.BYTESPIKE_API_KEY,
})
const resp = await client.images.generate({
  model: "seedream-v5lite",
  prompt: "An ink-wash mountain at dawn, soft mist",
  size: "1024x1024",
  n: 4,
})
resp.data.forEach((img) => console.log(img.url))
```

## 错误

| Code | 触发条件 | 计费？ |
|---|---|---|
| 400 / 401 / 402 / 403 | 标准 | 不计费 |
| 413 | prompt 过长 | 不计费 |
| 429 | 限流 | 不计费 |
| 451 | prompt 被上游安全过滤拦截 | 不计费 |
| 5xx | 上游问题 | 不计费（自动重试） |

## 何时选用

- prompt 迭代 / 设计探索 —— 还没决定要用 Seedream V4 或 V4.5 出终稿前。
- 大批量构思 —— 试十种变体的代价值得在意时。
- 美学 1024² 终稿，见 [Seedream V4](/zh/api-reference/image/seedream-v4)。
- 2048² + 文字边缘保真，见 [Seedream V4.5](/zh/api-reference/image/seedream-v4-5)。
