**厂商：** MiniMax
**Model ID：** `minimax-m2.5`
**能力：** 128K 上下文 · 工具调用 · 流式 · CJK 原生
**计价：** 按 token，中端档（[实时费率](https://bytespike.ai/pricing#text)）

MiniMax M2.5 是 MiniMax 的上一代生产模型。它凭借中文创意写作树立了口碑 ——
人物声音一致性、对话、叙事节奏 —— 在多数竞品语气会被压平的价位档里仍能保持
表现。对该细分场景，它仍可上生产。新一般用途项目优先选
[MiniMax M2.7](/zh/api-reference/text/minimax-m2-7)。

## Request

```bash
curl https://llm.bytespike.ai/v1/chat/completions \
  -H "Authorization: Bearer $BYTESPIKE_API_KEY" \
  -H "content-type: application/json" \
  -d '{
    "model": "minimax-m2.5",
    "messages": [{"role": "user", "content": "续写这段对话，保持角色性格。"}]
  }'
```

### Body 参数

| 字段 | 类型 | 必填 | 默认 | 说明 |
|---|---|---|---|---|
| `model` | string | 是 | — | `minimax-m2.5` |
| `messages` | array | 是 | — | 原生支持 CJK。 |
| `max_tokens` | integer | 否 | 模型上限 | 最大值：8192。 |
| `temperature` | number | 否 | 1.0 | 创意写作在 1.0–1.3 区间表现更好。 |
| `tools` | array | 否 | — | 支持 function calling。 |
| `stream` | boolean | 否 | false | SSE 流式。 |

## Response

```json
{
  "id": "chatcmpl-…",
  "object": "chat.completion",
  "model": "minimax-m2.5",
  "choices": [{"index": 0, "message": {"role": "assistant", "content": "..."}, "finish_reason": "stop"}],
  "usage": {"prompt_tokens": 287, "completion_tokens": 612, "total_tokens": 899}
}
```

## 代码示例

<CodeGroup>

```bash cURL
curl https://llm.bytespike.ai/v1/chat/completions \
  -H "Authorization: Bearer $BYTESPIKE_API_KEY" \
  -H "content-type: application/json" \
  -d '{"model": "minimax-m2.5", "messages": [{"role": "user", "content": "续写对话"}]}'
```

```python Python
from openai import OpenAI

client = OpenAI(base_url="https://llm.bytespike.ai/v1", api_key="$BYTESPIKE_API_KEY")
resp = client.chat.completions.create(
    model="minimax-m2.5",
    messages=[{"role": "user", "content": "续写这段对话，保持角色性格。"}],
)
print(resp.choices[0].message.content)
```

```javascript Node
import OpenAI from "openai"

const client = new OpenAI({
  baseURL: "https://llm.bytespike.ai/v1",
  apiKey: process.env.BYTESPIKE_API_KEY,
})
const resp = await client.chat.completions.create({
  model: "minimax-m2.5",
  messages: [{ role: "user", content: "续写这段对话，保持角色性格。" }],
})
console.log(resp.choices[0].message.content)
```

## 流式与缓存

`"stream": true` 走 SSE。自动 prompt caching。

## Errors

| Code | 触发条件 | 是否计费 |
|---|---|---|
| 400 / 401 / 402 / 422 / 429 | 标准 | 否 |
| 5xx | 上游 | 否（自动重试） |

## 何时选用

- 已针对这个版本验证过的现有创意写作流水线。
- 新项目优先选 [MiniMax M2.7](/zh/api-reference/text/minimax-m2-7)。
- 中文通用旗舰，见 [Kimi K2.6](/zh/api-reference/text/kimi-k2-6) 或 [GLM-5.1](/zh/api-reference/text/glm-5-1)。
